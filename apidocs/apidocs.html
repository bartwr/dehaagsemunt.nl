List all Events
GET/events{?keywords,location,date,category,within,units,sort_order,page_size,page_number}

Return a list of events bases on query parameters.

Example URI

GET https://api.vooot.nl/events?keywords=vergadering&location=The Hague&date=Today&category=&within=50&units=km&sort_order=date&page_size=50&page_number=0
URI Parameters
HideShow
keywords
string (optional) Example: vergadering

The search keywords

location
string (optional) Example: The Hague

A location name to use in filtering the search results. Locations in the form ‘San Diego’, ‘San Diego, TX’, ‘London’, ‘United Kingdom’, and ‘Calgary, Alberta, Canada’ are accepted, as are postal codes (‘92122’). Full addresses (‘1 Infinite Loop, Cupertino, CA’) and common geocoordinate formats (‘32.746682, -117.162741’) are also accepted, but the within parameter is recommended in order to set a search radius.

date
string (optional) Example: Today

Limit this list of results to the specified date range. Date ranges are specified by label. Currently supported labels include All, Future, Past, Today, Last Week, This Week, Next week, and months by name, e.g. October.

category
string (optional) 

Limit the search results to this category ID. A list of categories may be specified separated by commas. See /event/categories for a list of categories and their IDs.

within
integer (optional) Example: 50

If within is set and the location parameter resolves to a specific geolocation, the search will be restricted to the specified radius. If the location parameter does not resolve to a specific location, this parameter is ignored.

units
string (optional) Default: km Example: km

One of “mi” or “km”, the units of the “within” parameter.

Choices: km mi

count_only: (boolean, optional) - If count_only is set, an abbreviated version of the output will be returned. Only total_items and search_time elements are included in the result.
string (required) Default: false 

Choices: true false

sort_order
string (optional) Default: date Example: date

The sort order.

Choices: popularity date title relevance venue_name

sort_direction (string, optional) - One of 'ascending' or 'descending'. Default varies by sort_order.
string (required) 
page_size
number (optional) Default: 50 Example: 50

The desired number of results per page. Maximum: 50.

page_number
number (optional) Default: 0 Example: 0

The desired page number.
